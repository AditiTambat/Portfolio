Project

Honeyword & Password Based Authentication System

A Java-based web application designed to enhance security by using decoy passwords (honeywords) alongside real passwords to detect unauthorized access attempts. When a honeyword is entered, the system triggers an alert to protect the account. The system adds a salt to passwords before hashing them for additional security. The salted hashes for password storage ensure additional protection against breaches.

Features:
  • Secure user registration and login.
  • Honeyword detection to identify unauthorized access.
  • Alerts triggered on honeyword entry.
  • Password salting and hashing with BCrypt.
